# BetterSpeedometer
![Version](https://img.shields.io/badge/version-1.0.0-navy) <br>
A little handy mod that lets u see the horizontal, vertical and 
overall speed all at once!! 
Even configurable through the settings menu! <br>
---
### Note to users: 
This will lead to it being bugged when uninstalling 
due to how it technically uses an invalid pref value.
Just change the speedometer value in settings to fix it!
